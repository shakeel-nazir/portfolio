I created a TV show and made a broadcast package. I feel like this is a strong piece because I created a TV show from scratch and then made a full broadcast package for it. I created a lower third, bumper and show intro. I went through the full process when creating this package. I created the creative direction, made stills and mood boards.

When making this broadcast package I created a mood board that set the direction of the whole piece.

I created assets that I would use in the final broadcast package with the same theme as the mood board.

I created these still scenes with the assets using the same style as the mood board and so that I would have an idea what the whole motion piece would look like.
